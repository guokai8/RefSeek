## Highlights

### Search & Download
- Search across **PubMed**, **CrossRef**, **Semantic Scholar**, and **OpenAlex**
- Up to 200 results per query, sortable by relevance, year, citation count, or journal
- Structured queries: `author:Smith title:CRISPR year:2024`
- PDF download: Unpaywall → PMC → Sci-Hub
- Batch download: select multiple or download all at once

### Paper Library
- Tags, categories, notes, search/filter
- BibTeX export (single or all)
- PDF viewer integration

### AI Features (zero setup — works immediately)
- **Similar Papers**: Finds related papers in your library using Apple NLEmbedding
- **Quick Summary**: Extracts key sentences from abstracts or PDFs
- **Tag Suggestions**: AI keyword extraction
- **PDF Text Extraction**: Reads downloaded PDFs

### AI Features (optional — Ollama)
- **One-Click Install**: Set up Ollama from within the app (Settings → AI)
- No terminal, no API keys, no accounts needed
- Deep structured summaries (Key Findings, Methods, Significance, Limitations)
- LLM-powered smart tag suggestions

### Extras
- Menu bar quick access
- Global hotkey ⌘⇧R
- Right-click "Search in RefSeek" service
